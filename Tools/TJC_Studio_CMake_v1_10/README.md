# TJC_Studio v1.10.0

Génère un projet Arduino complet.

## Build

```cmd
build_vs2026_x64.bat
```

## Génération

```cmd
build\Release\TJC_Studio.exe examples\01_alias_test.HMI GeneratedUI generated
```

```text
generated/

├── GeneratedUI.h

├── GeneratedUI.json
├── README.md

```
